**Multiplayer2.5D** is a simple project built using [Construct 3](https://www.construct.net/) that serves as the foundation for a multiplayer semi 3D type of game. Currently is using the "Multiplayer game example" included in C3 as the base to create a more refined project.

---

<details>
<summary> üì• How to Use / Run </summary>


1. Open the project with **Construct 3** (https://editor.construct.net/).
2. Run the layout or preview the project.
3. Create a login to identify your player.
4. The game will load and will wait for other players to join.
5. To test in your own computer, open a different browser to connect to the same game.
6. Play.

> Make sure you have access to the internet or a licensed version of Construct 3 if using features that require it.
>> The basic versions of this projects should be under the free version restrictions, to allow anyone to check it out
